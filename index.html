Setup koneksi Google Sheets
1
Pasang script ke setiap file Google Sheets
Buka tiap file → Extensions → Apps Script → paste isi Code.gs → ubah SHEET_NAME sesuai nama tab → Deploy → Web app → Anyone → Deploy → copy URL.
2
Daftarkan URL tiap file (satu per bulan)
Tambahkan URL dan label bulan untuk tiap file. App akan membaca semua file sekaligus dan menggabungkan datanya.
3
Admin Key
Harus sama dengan ADMIN_KEY di script (default: rahasia123)
